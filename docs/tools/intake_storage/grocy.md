# Grocy

## What it is
Grocy is a self-hosted web-based groceries & household management solution for your home.

## What problem it solves
It brings ERP-style management to your home, helping you track stock, plan meals, and manage chores.

## Where it fits in the pipeline
**Store / Act**

## Typical use cases (in this homelab / family automation context)
- **Stock Management**: Keeping track of pantry items to avoid over-purchasing.
- **Meal Planning**: Planning weekly meals based on what is in stock.
- **Chore Tracking**: Assigning and tracking recurring household chores.

## Integration points
- **Home Assistant**: Full integration via an official add-on and dedicated entities.
- **REST API**: Allows for barcode scanners and custom interfaces to interact with the database.

## Licensing and cost
- **Open Source**: Yes (MIT)
- **Cost**: Free
- **Free tier**: N/A (Self-hosted)
- **Self-hostable**: Yes

## Strengths
- Extremely feature-rich (Barcodes, Recipes, Batteries, Chores).
- Strong Home Assistant integration.
- Active community.

## Limitations
- UI can be overwhelming and feels "corporate".
- Requires significant data entry to be useful initially.

## Alternatives / Related tools
- **Homebox** (for general inventory)
- **OurGroceries** (Proprietary)
