# Logseq

## What it is
Logseq is a privacy-first, open-source knowledge base that works as an outliner on top of local Markdown or Org-mode files.

## What problem it solves
It provides a non-linear way to capture thoughts and manage tasks, focusing on a daily journal workflow and bi-directional linking.

## Where it fits in the pipeline
**Store / Reason (Knowledge)**

## Typical use cases (in this homelab / family automation context)
- **Daily Journaling**: Capturing quick thoughts and family events as they happen.
- **Task Management**: Managing complex, nested tasks that are deeply integrated with your notes.
- **Privacy-First Personal Wiki**: Building a highly connected web of knowledge that remains local and encrypted.

## Integration points
- **Local File System**: Direct access to Markdown files for external processing.
- **Plugins**: Supports various community-built tools for calendar integration and data visualization.

- Native outliner experience.
- Strong focus on privacy and data ownership.
- Powerful query language for dynamic content.
