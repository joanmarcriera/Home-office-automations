# Perplexity

## What it is
Perplexity is an AI-powered search and answer engine that provides real-time information with cited sources.

## What problem it solves
It bridges the gap between a traditional search engine (which returns links) and a chatbot (which can hallucinate), providing verified information for research and troubleshooting.

## Where it fits in the pipeline
**Reason / Understand**

## Typical use cases (in this homelab / family automation context)
- **Technical Research**: Looking up the latest compatibility for a specific smart home device or server component.
- **Fact Checking**: Verifying school requirements or administrative procedures.
- **Productivity Comparison**: Researching the best tools for a specific automation task (e.g., comparing Vikunja vs. Todoist).

## Integration points
- **API**: Allows for integration into larger research pipelines or custom assistants.
- **Perplexity Pages**: For creating structured reports and documentation from search results.

## Licensing and cost
- **Open Source**: No
- **Cost**: Freemium
- **Free tier**: Yes (Standard search).

## Strengths
- High accuracy with direct citations to sources.
- Real-time access to the live internet.
- Clean and efficient interface for deep research.

## Limitations
- Less focus on creative text generation compared to ChatGPT.
- Accuracy still depends on the quality of the sources found.
