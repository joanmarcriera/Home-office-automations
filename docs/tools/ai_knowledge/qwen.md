# Qwen

## What it is
Qwen is a series of Large Language Models (LLMs) developed by Alibaba Cloud, including general-purpose, coding, and vision models.

## Typical use cases
- General purpose chat and reasoning
- Code generation and assistance
- Multimodal understanding
