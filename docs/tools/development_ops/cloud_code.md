# Cloud Code

Cloud Code is a set of IDE plugins for popular IDEs (VS Code, IntelliJ) to help you develop, deploy, and debug cloud-native applications.

## Description
It provides tools for working with Kubernetes, Google Cloud, and other cloud providers directly from your IDE.
