# GPT Engineer

## What it is
GPT Engineer is an AI tool that builds entire codebases based on a single natural language prompt. It asks clarifying questions and then generates the folder structure and files.

## What problem it solves
Drastically reduces the time from idea to a working prototype by handling the initial setup and boilerplate generation of a project.

## Where it fits in the pipeline
**Act (Development)**

## Typical use cases (in this homelab / family automation context)
- Bootstrapping a new web-based dashboard for home metrics.
- Generating a complete scaffold for a new microservice in the automation stack.

## Integration points
- **CLI**: Primary interface is via the terminal.
- **GitHub**: For saving generated projects.

## Licensing and cost
- **Open Source**: Yes (MIT)
- **Cost**: Free (Software) + LLM costs.
- **Free tier**: N/A
- **Self-hostable**: Yes

## Strengths
- High-speed project creation.
- Interactive clarification process.
- Open and extensible.

## Limitations
- Best for starting projects; harder to use for long-term maintenance of large existing codebases.
