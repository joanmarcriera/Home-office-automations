# Tabnine

## What it is
Tabnine is an AI assistant for software developers that provides context-aware code completions and a chat interface.

## What problem it solves
One of the earliest AI assistants, it focuses on privacy and security, allowing for isolated deployments that don't share code with third parties.

## Where it fits in the pipeline
**Act (Development)**

## Typical use cases (in this homelab / family automation context)
- Offline-ready code completions for secure homelab environments.
- Consistent AI assistance across many different text editors.

## Integration points
- **IDE Extensions**: Very broad support, including VS Code, Vim, Emacs, and older JetBrains versions.

## Licensing and cost
- **Open Source**: No
- **Cost**: Paid (Subscription)
- **Free tier**: Yes (Basic autocomplete)
- **Self-hostable**: Yes (Enterprise only)

## Strengths
- Strong privacy focus.
- Wide IDE compatibility.
- Hybrid local/cloud processing option.

## Limitations
- Autocomplete quality can sometimes lag behind the latest LLM-based tools.
- Full features are paid.
