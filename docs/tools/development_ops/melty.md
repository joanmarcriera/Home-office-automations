# Melty

## What it is
Melty is an open-source AI code editor that aims to be the "first AI editor that actually works like a human". It watches you code and helps you when you're stuck.

## What problem it solves
It focuses on a tighter feedback loop between the human developer and the AI, reducing the "request-response" friction of traditional chat-based tools.

## Where it fits in the pipeline
**Act (Development)**

## Typical use cases (in this homelab / family automation context)
- Pair programming on complex home automation logic.
- Getting real-time help with unfamiliar libraries or APIs.
