# Superconductor

## What it is
Superconductor is a mobile-friendly AI agent orchestrator that allows developers to launch and manage parallel AI coding agents from a phone or tablet.

## What problem it solves
It addresses the need for managing long-running AI coding tasks on the go, providing a mobile UI to review changes and receive notifications as agents complete tickets.

## Where it fits in the pipeline
**Act (Development / Orchestration)**

## Typical use cases (in this homelab / family automation context)
- Monitoring a long-running refactoring task from a mobile device.
- Reviewing AI-generated pull requests for home automation scripts while away from the workstation.

## Integration points
- **GitHub**: For repository access and PR management.
- **HTTP Preview**: Built-in ability to view web services launched by agents.
